# 模型下载的若干问题

CapsWriter-Offline 完全离线运行，所有模型都在本地：

| 引擎名 | 准确性 | 速度 | 格式 | 调用方式 | 显卡加速 |
|------|-------|------|------|---------|---------|
| Paraformer | ★★★☆☆ | ★★★★★ | ONNX | sherpa_onnx | 不支持 |
| SenseVoice-Small | ★★★☆☆ | ★★★★★ | ONNX | 自定义引擎 | 可加速 |
| Fun-ASR-Nano | ★★★★☆ | ★★★★☆ | ONNX + GGUF | 自定义引擎 | 可加速 |
| Qwen3-ASR | ★★★★★ | ★★★☆☆ | ONNX + GGUF | 自定义引擎 | 可加速 |

Paraformer 通过 sherpa_onnx 调用，不支持显卡加速，且需要外挂标点符号模型，不建议使用，仅作纪念性保留。

性能差的电脑，建议用 SenseVoice-Small；没有独显的电脑可以尝试用 Fun-ASR-Nano 和 Qwen3-ASR，如果延迟可以接受就用；有独显的电脑直接上最顶级的 Qwen3-ASR，准确率夯爆。

## 三、如何选择引擎

在 [`config_server.py](../config_server.py) 中设置：

```python
model_type = 'fun_asr_nano'    # 可选：fun_asr_nano, sensevoice, qwen_asr, paraformer
```

Paraformer 不具备标点符号能力，需要外挂标点符号模型 Punct-CT-Transformer；Qwen3-ASR 不具备时间戳能力，需要外挂时间戳模型 ForcedAligner 才能生成时间准确的字幕。
